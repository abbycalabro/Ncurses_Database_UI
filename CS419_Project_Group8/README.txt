****************
* Installation *
****************

Install Python 2.7 and PostgreSQL according to your operating system's instructions:

python: 		https://docs.python.org/2/install/
postgreSQL:		http://www.postgresql.org/download/

If using windows, install the psycopg2 and npyscreen libraries according the following instructions:

pyscopg2:		http://initd.org/psycopg/docs/
npyscreen:		https://pypi.python.org/pypi/npyscreen

Or to install psycopg2 and npyscreen using Linux or OS X, enter the following on the command line:

pip install -r requirements.txt

***************
* Run program *
***************

To execute the program, run the following command:

python interface.py
